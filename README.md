# ELore

Prosta wersja serwisu moodle. Umożliwia:
 - założenie konta
 - założenie własnego kursu
 - zarządzanie kursem (dodawanie, usuwanie członków lub lekcji)
 - zarządzanie użytkownikami (administrator)

 Użyte technologie to:
    - Angular
    - Node .js
    - Express .js
    - Mongoose
    - Bootstrap
